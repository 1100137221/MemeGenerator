# Introduction
 建立一個 Meme 圖片產生器的網頁, <br/>
 並且可以在圖片上下添加文字, <br/>
 練習使用 react + redux 來建立這個網站  

# How to

## Setup Basics

1. npm install
2.  go to https://imgflip.com/ register member, <br/>
    and to this file: /src/actions/secrets.js,  input your username and password.
2. 打開 http://localhost:3000
